### [Blog 1 - Building a Serverless Supply Chain Management Solution for Automotive Customers with AWS AppSync and Amazon Aurora Serverless](3.1-Blog1/)

This blog introduces a serverless architecture for building a supply chain management system in the automotive industry using AWS services. It explains how **Amazon Cognito**, **AWS AppSync**, **Amazon Aurora Serverless**, and **AWS Secrets Manager** work together to build a scalable GraphQL API, process real-time data, and securely manage database credentials. The article also highlights the benefits of using JavaScript Resolvers in AppSync to reduce latency, improve application performance, and optimize operational costs.

### [Blog 2 - Customize Federated Sign-in with the New Amazon Cognito Inbound Federation Lambda Trigger](3.2-Blog2/)

This blog introduces the **Inbound Federation Lambda Trigger** feature in Amazon Cognito, which enables developers to customize the authentication flow when users sign in through third-party Identity Providers such as Google, Facebook, OIDC, or SAML. It explains how the trigger can be used to normalize user attributes, automatically link duplicate accounts, remove unnecessary attributes, and improve identity management for both B2B and B2C applications.

### [Blog 3 - Using Generative AI for Proactive Infrastructure as Code (IaC) Generation that's Compliant with Service Control Policies Using Amazon Bedrock](3.3-Blog3/)

This blog presents a solution that leverages **Amazon Bedrock** and Generative AI to automatically generate **Infrastructure as Code (IaC)** from natural language requests while ensuring compliance with an organization's **Service Control Policies (SCPs)**. It also describes an architecture that integrates **AWS CloudTrail**, **Amazon EventBridge**, **AWS Lambda**, **Amazon S3**, **Amazon Cognito**, and **AWS WAF** to automatically synchronize policy updates, strengthen security, and support efficient cloud infrastructure deployment.
