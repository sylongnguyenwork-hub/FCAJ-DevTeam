### [Blog 1 - Building a Serverless Supply Chain Management Solution for Automotive Customers with AWS AppSync and Amazon Aurora Serverless](3.1-Blog1/)

Blog này giới thiệu giải pháp xây dựng hệ thống quản lý chuỗi cung ứng cho ngành công nghiệp ô tô bằng kiến trúc **serverless** trên AWS. Bài viết trình bày cách kết hợp **Amazon Cognito**, **AWS AppSync**, **Amazon Aurora Serverless** và **AWS Secrets Manager** để xây dựng GraphQL API có khả năng mở rộng tự động, xử lý dữ liệu theo thời gian thực và bảo mật thông tin kết nối cơ sở dữ liệu. Đồng thời, bài viết cũng phân tích lợi ích của việc sử dụng JavaScript Resolvers trong AppSync nhằm giảm độ trễ, tối ưu hiệu năng và tiết kiệm chi phí vận hành.

### [Blog 2 - Customize Federated Sign-in with the New Amazon Cognito Inbound Federation Lambda Trigger](3.2-Blog2/)

Blog này giới thiệu tính năng **Inbound Federation Lambda Trigger** của Amazon Cognito, cho phép can thiệp vào luồng xác thực khi người dùng đăng nhập thông qua các Identity Provider như Google, Facebook, OIDC hoặc SAML. Bài viết trình bày cách chuẩn hóa dữ liệu người dùng, tự động liên kết các tài khoản trùng lặp (Account Linking), loại bỏ các thuộc tính dư thừa và nâng cao khả năng quản lý danh tính trong cả môi trường B2B và B2C.

### [Blog 3 - Using Generative AI for Proactive Infrastructure as Code (IaC) Generation that's Compliant with Service Control Policies Using Amazon Bedrock](3.3-Blog3/)

Blog này trình bày giải pháp sử dụng **Amazon Bedrock** và Generative AI để tự động sinh mã **Infrastructure as Code (IaC)** từ yêu cầu bằng ngôn ngữ tự nhiên, đồng thời đảm bảo mã được tạo ra luôn tuân thủ **Service Control Policies (SCPs)** của tổ chức. Bài viết cũng giới thiệu kiến trúc tích hợp AWS CloudTrail, Amazon EventBridge, AWS Lambda, Amazon S3, Amazon Cognito và AWS WAF nhằm tự động cập nhật chính sách, tăng cường bảo mật và hỗ trợ triển khai hạ tầng đám mây hiệu quả.
