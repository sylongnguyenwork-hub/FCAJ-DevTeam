#### Serverless Backend with Lambda

In this section, you will configure and deploy the core logic processing layer (Serverless Backend) using **AWS Lambda**. The automated Intelligent Document Processing (IDP) system will utilize Lambda functions running on the Python 3.12 environment to receive messages, extract text using AI, and serve data to the user interface via APIs.
